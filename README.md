# HA Solutions Website

A complete replication of the Smoothstack website, rebranded as HA Solutions. This website showcases technology talent development and workforce transformation services.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Interactive Elements**: Dropdown menus, popup dialogs, and smooth scrolling
- **Accessible**: Skip links and proper ARIA labels for better accessibility
- **Cross-browser Compatible**: Works on all modern browsers

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - Complete CSS styling for the website
- `script.js` - JavaScript for interactivity and animations
- `README.md` - This file

## How to Use

Simply open `index.html` in your web browser to view the website. No additional setup or build process required.

## Sections

- Hero Section with call-to-action buttons
- Why Work with Us section
- Transformation showcase
- Industry insights and webinars
- Technology alliances
- Mission statement
- Industry growth statistics
- Contact and CTA sections
- Footer with links and social media

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

To customize the website:
- Edit `styles.css` to change colors, fonts, and layout
- Modify `index.html` to update content and structure
- Adjust `script.js` to add or modify functionality
